# **App Name**: SocraticAI

## Core Features:

- Chat Interface: Real-time chat for student-tutor interaction, facilitating question asking and guidance.
- Problem Input: An input area for students to enter problems or questions they need assistance with.
- Socratic Questioning: AI-driven questioning engine that guides students towards solutions using the Socratic method.
- Hint System: Progressive hint delivery system that starts with general advice and becomes more specific as needed, helping students to solve problems independently. The tool uses LLM to decide what information to add, if necessary.
- Step-by-Step Tracker: Visual representation of student progress through a problem, breaking down complex tasks into manageable steps.
- Learning Recap: Summarize the key learnings from the session.

## Style Guidelines:

- Primary color: Hunter green (#355E3B) for headers, navigation, and primary buttons, evoking an academic and trustworthy feel.
- Accent color 1: Gold (#D4AF37) for highlights, progress indicators, or success states, providing visual cues.
- Accent color 2: Beige/cream (#F5F5DC) for main content areas to reduce eye strain, creating a comfortable learning environment.
- Body and headline font: 'Literata', a transitional serif with a literary and intellectual feel.
- Use simple, academic-style icons to represent concepts and tools.
- Clean, structured layout with clear visual hierarchy to promote ease of navigation and comprehension.
- Subtle animations and transitions to provide feedback and maintain user engagement without being distracting.
